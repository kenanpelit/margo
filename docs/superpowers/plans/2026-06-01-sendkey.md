# sendkey Implementation Plan

> REQUIRED SUB-SKILL: superpowers:executing-plans. Steps use `- [ ]`.

**Goal:** `sendkey,<combo>[,<appid-regex>][,<fallback>]` dispatch action — inject a synthetic key combo into the focused window, optionally gated by focused app_id with a fallback action.

**Architecture:** Pure combo/key-name/regex helpers in `margo/src/dispatch/sendkey.rs` (unit-tested) + an impure `MargoState::send_key`/`inject_combo` that drives the seat keyboard (`keyboard.input(..., FilterResult::Forward)`), mirroring `handle_keyboard`. Wired via a `"sendkey"` arm in `dispatch/mod.rs`.

---

### Task 1: pure helpers in `dispatch/sendkey.rs` (TDD)
**Files:** Create `margo/src/dispatch/sendkey.rs`; `mod sendkey;` in `dispatch/mod.rs`.
- [ ] Tests: `parse_combo("ctrl+Tab")==Some(KeyCombo{mods:[29],key:15})`; `parse_combo("ctrl+shift+Tab")` mods `[29,42]`; `parse_combo("Tab")` mods `[]`; `parse_combo("ctrl+bogus")==None`; case-insensitive modifiers + key (`CTRL+tab`). `key_name_to_evdev("Page_Up")==Some(104)`, `"Prior"==104`, `"F5"==63`, `"nope"==None`. `appid_matches(Some("Kenp"),"^(Kenp|Ai)$")==true`, `(Some("kitty"),"^Kenp$")==false`, `(None,"^x$")==false`, bad regex → false. `parse_fallback("focusdir:up")==Some(("focusdir","up"))`, `"zoom"==Some(("zoom",""))`.
- [ ] Implement: `struct KeyCombo{mods:Vec<u32>,key:u32}`; `MOD_EVDEV` map (ctrl/control→29, shift→42, alt/meta?no→56, super/logo/mod→125); `key_name_to_evdev` table (layout-independent set from the spec); `parse_combo` (split `+`, last=key via key_name_to_evdev, rest=mods via MOD_EVDEV, any miss→None); `appid_matches` (regex::Regex::new ok + is_match); `parse_fallback(&str)->Option<(&str,&str)>` (split_once ':' or whole=action,arg="").
- [ ] `cargo test -p margo sendkey` green. Commit.

### Task 2: injection on MargoState
**Files:** `margo/src/dispatch/sendkey.rs` (impl block).
- [ ] `impl MargoState { pub fn send_key(&mut self, arg:&Arg); fn inject_combo(&mut self, combo:&KeyCombo); }`.
  - `send_key`: `let Some(combo)=parse_combo(arg.v.as_deref().unwrap_or(""))` else warn+return. If `arg.v2` Some & non-empty: `let app = self.focused_app_id();` if `!appid_matches(app.as_deref(), regex)` → run fallback (`arg.v3`) via `parse_fallback` → build sub `Arg{ v: (!a.is_empty()).then(|| a.to_string()), ..Default::default() }`, guard `action!="sendkey"`, `crate::dispatch::dispatch_action(self,action,&sub)`; return. Else fall through to inject.
  - `inject_combo`: `let Some(kb)=self.seat.get_keyboard() else return;` build sequence: mods press (in order), key press, key release, mods release (reverse); for each `(evdev,state)`: `let serial=SERIAL_COUNTER.next_serial(); let time=self.now_ms(); kb.input::<(),_>(self,(evdev+8).into(),state,serial,time,|_,_,_| FilterResult::Forward);`
  - `focused_app_id(&self)->Option<String>`: `self.focused_client_idx().map(|i| self.clients[i].app_id.clone())` (confirm field/method names at impl; reuse the snapshot's `c.app_id`).
  - `now_ms()`: monotonic ms — reuse margo's clock if `state` has one, else `std::time::Instant`-based or a CLOCK_MONOTONIC read. (Check for an existing `state.clock`/`start_time`; pick the established pattern.)
- [ ] `cargo build -p margo`. Commit.
